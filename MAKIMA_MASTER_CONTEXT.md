# MAKIMA Website --- Master Context

## Project Vision

Create a premium editorial tribute website for Makima. It should feel
like a luxury magazine, not an anime fan site.

## Inspiration

-   Shopify Editions
-   Dia Browser
-   Apple product pages
-   Awwwards
-   GTA VI official website

## Visual Language

Editorial • Luxury • Minimal • Cinematic • Japanese • High-fashion •
Sophisticated

## Colors

-   Background: #F8F6F2
-   Secondary: #F3EEE8
-   Surface: #FFFFFF
-   Accent: #A63D3D
-   Text: #111111
-   Muted: #7C7C7C
-   Hair Accent: #D7B26A

## Grid

-   12 columns
-   1440px content width
-   8pt spacing system
-   Large whitespace

## Typography

-   Large editorial serif headings
-   Clean sans-serif body
-   Oversized hero typography

## Motion

-   Lenis smooth scrolling
-   GSAP + Framer Motion
-   Soft easing
-   Staggered reveals

## Site Structure

1.  Hero
2.  Sticky Storytelling
3.  Character Profile
4.  Relationships
5.  Timeline
6.  Quote
7.  Gallery
8.  Legacy

## Hero

-   Full viewport
-   Cinematic Makima image/video
-   Hidden navigation
-   Scroll indicator
-   Massive:
    -   マキマ
    -   MAKIMA
    -   Public Safety Devil Hunter

## Sticky Story

Left sticky text, right changing editorial content.

## Character Profile

Editorial split layout with portrait and metadata.

## Relationships

Luxury dossier/book experience with Denji, Aki, Power, Kishibe and
Pochita.

## Timeline

Horizontal milestone cards.

## Quote

Single full-screen editorial quote.

## Gallery

Editorial masonry with mixed image sizes.

## Footer

Minimal legacy section.

## Never Do

-   Neon
-   Dashboard UI
-   Gaming UI
-   Generic AI layouts
-   Heavy gradients
-   Clutter
-   Repetitive sections

## Final Goal

Create an Awwwards-quality editorial experience.
